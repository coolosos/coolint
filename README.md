# coolint
This package provides lint rules for Dart and Flutter.

## Usage

### 1. Add dependency to `pubspec.yaml`

```yaml
dev_dependencies:
  coolint: ^2.0.1
```

###  2. Add configuration to `analysis_options.yaml`

```yaml
include: package:coolint/coolint.yaml
```
